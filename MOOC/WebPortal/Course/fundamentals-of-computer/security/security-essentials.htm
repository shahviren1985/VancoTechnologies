Microsoft Security Essentials is free antimalware software for your computer. It helps protect against viruses, spyware, and other malicious software. It's available as a no-cost download for Windows 7, Windows Vista, and Windows XP SP2 and higher.

Microsoft Security Essentials is always kept up to date, runs quietly in the background, and schedules scans when your computer is most likely idle. You only see alerts when you need to take action.
