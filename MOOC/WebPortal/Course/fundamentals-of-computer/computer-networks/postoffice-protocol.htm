A protocol is a set of rules to govern the data transfer between the devices. The rules are used for the following purposes.

  • To compress the data
  • To indicate that sender has finished sending a message
  • To indicate that receiver has received a message

Real life example of protocol is: conversation between two people.

Person A
Person B
I know English. Can we talk?
I know French & English. Yes we talk?
[Since ‘Person B’ knows English they can talk.] Shall I start conversation in English?
Yes, I know English. Yes you can start.
Actual Conversation begins and continues
Actual Conversation begins and continues
Conversation is ended by Person A saying Good bye
Conversation ended. [‘Person B’ now available to interact with another person]

Sample conversation protocol



This is a communication protocol that computers use to communicate over a network. This protocol first establishes the connection between sender and receiver before starting a communication.

E.g. when you try calling someone, the connection is established. Once connection is established then you can have a conversation.

There are different types of Protocol such as :

  • Transmission control Protocol (TCP)
  • Internet Protocol (IP)
  • Post office Protocol (POP)
  • Simple mail transport Protocol (SMTP)
  • Hyper Text Transfer Protocol (HTTP)
  • File Transfer Protocol (FTP)
