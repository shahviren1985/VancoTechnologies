To open an existing notepad document click on File > Open...


File Open dialog


Notepad will ask you if you want to save current document. The Open window appears (note the word 'Open' in the title bar) which allows you to browse to your existing document.

Note: the File name has a *.txt which means when you browse for document, only files ending with “.txt” will show up in the Open window.
