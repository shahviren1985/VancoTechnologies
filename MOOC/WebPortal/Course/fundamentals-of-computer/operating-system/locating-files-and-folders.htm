If user doesn’t know where a document or folder is located, user could use the Search companion to locate it in the local storage such as Hard Disk, CD.

To search for a file or folder

  • Click Windows button, and then type file or folder name you want to search in the text field.
  • Or open My Computer, In that right hand side corner there is text field of Search Computer. In that you can type file or folder name that you want to search.

Search files and folders


Once user has finished specifying the search criteria, user could click on Search to locate the file. Similarly user could search for specific information on Internet, or people in address book, search for a printer or a computer.
