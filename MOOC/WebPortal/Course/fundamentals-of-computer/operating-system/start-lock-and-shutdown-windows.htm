Start the computer :
  • Press the power button which is on CPU.
  • If password is set then enter the password to start your computer.
How to lock the windows :
  • Press Alt+Ctrl+Del button together.
  • Enter your password to unlock the windows.
How to Shutdown windows :
  • Before shutting down windows, save all the applications because it will close all applications while shutdown
  • Press Start button on the taskbar.
  • Click on Shutdown option. It will power off the computer.
