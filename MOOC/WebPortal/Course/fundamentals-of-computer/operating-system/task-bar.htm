When user logs into the Windows 7, task bar appears on the bottom of the screen. By default it remains visible when Windows 7 is running. Whenever any new program is started or a new window is opened, an icon representing the program appears on the taskbar.

To switch between windows or program, click on the icon representing the window on the task bar. When the program or window is closed, the icon disappears from the task bar.
