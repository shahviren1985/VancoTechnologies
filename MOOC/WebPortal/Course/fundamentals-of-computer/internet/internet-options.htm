Each of the browsers provides additional options to make browsing experience safe, secure, private and efficient. Use following steps to open the internet options in different web browsers:

Google Chrome

  1. Open Google Chrome browser window
  2. Use one of the following options :
    1. Enter following URL : chrome://settings/
    2. Click on small rectangle icon next to the address bar. It will open one menu. Click on “Settings” option.

Mozilla Firefox

  1. Open Mozilla Firefox browser window
  2. Press “Alt” + “T” keys together.
  3. Click on “Options” menu item.

Microsoft Internet Explorer

  1. Open Microsoft Internet browser window
  2. Press “Alt” + “T” keys together.
  3. Click on “Internet Options” menu item

Google Chrome browser window


You could perform many tasks using this internet options. Few of the tasks are as mentioned below :

  • Change the default web page of web browser
  • Change the security settings
  • Clear the browsing history
  • Set the content privacy options
  • Few advance options
