When you visit any of the websites, it is tracked in web browser to maintain it as history. User can clean this history by following the steps:

Google Chrome

  1. Open Google Chrome browser window
  2. Use one of the following options :
    1. Enter following URL : chrome://history/
    2. Click on small rectangle icon next to the address bar. It will open one menu. Click on “Settings” option. One new window will get opened. Click on “History” link on top-left side.

Google chrome – clean history option


Mozilla Firefox

  1. Open Mozilla Firefox browser window
  2. Press “Alt” + “T” keys together.
  3. Click on “Options” menu item.
  4. Click on “Advanced” tab in new popup window
  5. Click on “Network” tab. You will get options to clear the cached web content as well as offline web content.

Firefox history options


Microsoft Internet Explorer

  1. Open Microsoft Internet browser window
  2. Press “Alt” + “T” keys together.
  3. Click on “Internet Options” menu item
  4. Inside “General” tab, you will find button “Delete…” to clear the history.
