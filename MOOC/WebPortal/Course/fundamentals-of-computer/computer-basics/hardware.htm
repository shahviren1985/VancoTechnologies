Computer hardware is the collection of physical elements that make a computer system. Computer hardware refers to the physical parts or components of a computer such as

  1. Monitor: screen used to display instruction and output
  2. Keyboard: Typing device used to input instructions
  3. Central Processing Unit (CPU) : Unit used for processing the input and the data
  4. Mouse etc.: hand controlled device used on a point and click interface
