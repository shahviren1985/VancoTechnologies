A CPU also makes use of Read Only Memory (ROM), in primary storage. This memory is used to load the initial computer programs. Small programs called firmware are often stored in ROM chips on hardware devices, and they contain instructions to perform most basic operations required to operate hardware devices. ROM memory cannot be easily or quickly overwritten or modified.
