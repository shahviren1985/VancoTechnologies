Professional looking user documents could be created formatted and edited easily using MS-Word 2007. MS Word 2007 have new user interface with new component called “Ribbon”. This “Ribbon” groups the similar tasks in same “Tab” e.g. Home, Insert etc. Each of these tabs has related options. E.g. Home tab have most frequently used tools such as text formatting, text styling etc.


Word Ribbon Options


  • Lots of preformatted templates are available with Word 2007. You can quickly create professional looking documents by choosing cover page, header & footer using these templates.
  • Charts & diagrams could be created in Word 2007 using 3D shapes, transparency, drop shadows & other effects.
  • Document authenticity, integrity and origin can be ensured using digital signature.
  • Word 2007 document can be exported to PDF file format
