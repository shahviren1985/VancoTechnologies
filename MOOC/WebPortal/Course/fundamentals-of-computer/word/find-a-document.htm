Many times you forget the exact name of word document. You might remember part of name of the document. In such cases, you can locate the exact file easily.

Follow these steps to find files :

  1. Select Office Button Open command on the menu bar. The Open dialog box will be displayed.
  2. In Look in: box, select the folder or click where you suspect your file will be located.
    1. Type few letters you remember that match with file name in File Name: box. If the typed letters are beginning letters of your file name, the file name will appear in the File Name: box.
    2. Click Open to open the document.
