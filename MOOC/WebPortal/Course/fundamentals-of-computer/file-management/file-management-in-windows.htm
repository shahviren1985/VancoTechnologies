Windows explorer is basic tool for “file management”. Windows Explorer displays the hierarchical list of files, folders, and storage drives on your computer. Windows Explorer can be used to copy, move, rename, and search for files and folders. For example, to copy a file, you can open a folder that contains the desired file to be copied and then just drag and drop the file to target folder or drive.


Windows Explorer


When files or folders are deleted from hard disk, Windows places them in the Recycle Bin. Each of the deleted files & folders could be retrieved from the Recycle Bin until the Recycle Bin is made empty. Files or folders deleted from a removable storage media (e.g. pen drive) are permanently deleted and are not sent to the Recycle Bin.
