Windows provides a quick and easy way to install new software programs. Add or Remove Programs utility in Control Panel help to manage programs and components on the computer system. This utility could be used to add/remove Windows components.

Add/remove programs


There are multiple ways to open this add/remove dialog box. One of the shortest methods is as mentioned below:

  • Click on Windows start button, click on “Run” option
  • Type “appwiz.cpl” in run dialog box
  • Press enter key or click on OK button
