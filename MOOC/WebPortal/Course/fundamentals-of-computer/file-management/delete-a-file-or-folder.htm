  • Click on Windows start button, and then click on My Documents
  • Click on the file or folder you want to delete
  • Under File, click Delete
  • Files or folders can also be deleted by right-clicking the file or folder and then clicking Delete
  • Deleted files or folders are stored in the Recycle Bin, until they are permanently removed from the Recycle Bin
  • To retrieve a deleted file, double-click the Recycle Bin icon on the desktop. Right-click on the file to retrieved, and then clicks on “Restore” option.
