A keyboard is the most common input device. Several kinds of keyboards are available, but they resemble each other with minor variations. The keyboard in most common use is the QWERTY board.

Each of the keyboards has different keys as listed below :

  1. Function keys – keys having name as F1 to F12
  2. Num pad – most of the key boards have Num pad on right side. However many laptop not have Num pad
  3. Special characters – each of the numeric keys have special characters attached to it. When user presses Shift Key + Any of the numeric key (1 to 0)‚ user will see the special characters.
  4. Alphanumeric characters – key board have alphabets A to Z. by default these keys are in lower case. Whenever user wants to have upper case letters, two options are available.
    1. Press CAPS Lock key. This key will generate upper case letters.
    2. Press Shift + alphabet key will generate upper case letters
  5. Numeric characters
  6. Special keys – apart from special characters present on numeric keys, there are few characters which are having dedicated keys. E.g. comma (‚) semi colon (;) etc.
