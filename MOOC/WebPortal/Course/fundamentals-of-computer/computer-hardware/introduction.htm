Computer hardware is the collection of physical devices that constitutes a computer system. Computer hardware refers to the physical parts or components of a computer such as monitor, keyboard, computer data storage, mouse, system unit (graphic cards, sound cards, memory, motherboard and chips), etc. all of which are physical objects that can be touched.

In this chapter we will learn about different type of hardware devices.

Different type of hardware devices are as follows:

  1. Input Devices
  2. Output Devices
