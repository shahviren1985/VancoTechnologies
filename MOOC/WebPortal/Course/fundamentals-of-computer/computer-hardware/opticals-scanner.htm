Many times user wants to input data into computer automatically. There are kind of devices which completely eliminate manual input of data. Few devices are mentioned below:

  1. Bar code reader
  2. Image scanner
  3. Optical character recognition (OCR)

Bar code reader – you must have observed bar codes in super markets. The bar-code reader is a special type of image scanner. An image scanner translates printed images into an electronic format that can be stored in a computer’s memory.

Image scanner – this device are used to automatic data collection. The devices of this category completely eliminate manual input of data. An image scanner translates printed images into an electronic format that can be stored in a computer’s memory, and with the right kind of software, one can alter a stored image.

Optical character recognition (OCR) - optical character recognition (OCR) device, used by banks to convert the scanned image of a typed or printed page into text that can be edited on the computer.
