Application software is the program that helps you perform useful tasks on a computer. An accounting software, enterprise software, graphics software, media players, and office suites etc. are few example of application software.

NOTEPAD

Notepad is a handy program that a user can type in text quickly and easily. It is not for publishing a book but more of a scratchpad. This chapter will cover following topics on Notepad :

  • Opening Notepad
  • Saving a document
  • Printing a document
  • Opening an existing document
