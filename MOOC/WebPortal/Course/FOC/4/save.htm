  • To save your work click on File > Save.
  • If you are saving the content for first time, save screen will appear. Enter file name and click on save button.
  • If content was previously saved, it will not ask you for file name. It will automatically save the content in same file.
  • You could directly press CTRL + S keys on your keyboard to save the content of your notepad.
