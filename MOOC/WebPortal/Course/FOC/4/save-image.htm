Image could be saved in many popular formats. Few of the popular formats are :

  • Bitmap (bmp)
  • JPEG (JPG)
  • Portable Network Graphics (PNG)

Preferred saving format is JPEG. JPG format is widely accepted, the image size small compare to other formats and easy to transfer.

Image can be saved by following the steps :

  • Click on File menu option.
  • Click on “Save” option
  • Choose “Save as type” option at the bottom of popup screen.
  • Give appropriate file name
  • Click on “Save” button
