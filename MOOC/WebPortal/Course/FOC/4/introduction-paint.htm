Application software is the program that helps you perform useful tasks on a computer. An accounting software, enterprise software, graphics software, media players, and office suites etc. are few example of application software.

MS PAINT

Paint is a program used to draw, color, and edit pictures. You can use Paint like a digital sketchpad to make simple pictures or add text and designs to other pictures, such as those taken with your digital camera.

MS Paint application could be started using following steps :

  1. Click on Windows start button
  2. Click on Programs and then accessories
  3. Click on Paint
