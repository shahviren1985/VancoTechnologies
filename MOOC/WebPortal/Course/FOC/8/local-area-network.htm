A network which is limited to a geographic area such as a writing lab, school, or building is known as Local Area Network (LAN).

Computers connected to a network are broadly categorized as :

  • Servers
  • Workstations

Servers are generally not used by humans directly, but rather run continuously to provide "services" to the other computers on the network. Services provided can include printing and faxing, file storage and sharing, messaging and many others. Servers tend to be more powerful than workstations.

Workstations have a human user which interacts with the network through computer/mobile devices. Workstations are any device through which human user can interact to utilize the network resources.
