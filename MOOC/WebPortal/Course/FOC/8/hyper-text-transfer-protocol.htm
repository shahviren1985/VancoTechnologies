HTML is language used to create hyperlinks and hypertext[e.g.: Links on a website]. Hyperlinks provide links between different resources (images, videos, text, animations etc.). HTTP is used to transfer a hypertext between two or more computers.

HTTP is based on the client/server principles. HTTP allows a client to establish connection with a server and make a request. The server accepts the connection initiated by the client and sends back the response.
