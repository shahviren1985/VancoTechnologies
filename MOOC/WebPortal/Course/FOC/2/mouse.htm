A mouse is a pointing device. It can perform functions like selecting menu commands, moving icons, resize windows, starting programs, and choosing options.

Now a day’s Optical or laser mouse is used to detect the movement. All windows based applications today are designed to work with a mouse. A mouse is used to replace hard-to-remember key combinations with easier "Point and Click" actions. However, it cannot substitute all keyboard operations. It can be alternative for commands based operations.

Mouse has three basic functions: right click, left click & scroll.

Right click – whenever user right clicks on screen, it shows the context menu. This context menu has additional options.

Left click – whenever user left clicks on screen, the item below the mouse pointer gets highlighted. User could perform additional operations on an item after it is highlighted.

Scroll – whenever the container (folders/web page etc.) has lots of items/content which could not be displayed within available display area, scroll bar appears on screen. The scroll present on mouse could be used to scroll up or scroll down.
