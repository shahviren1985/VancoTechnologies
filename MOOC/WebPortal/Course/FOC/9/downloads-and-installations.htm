Many a times you install unnecessary software, toolbars, browser extensions etc. These additional software application impacts the browser performance. Hence it is recommended that you should not install any of the additional software applications (unless it is absolutely necessary)

Toolbars

Most commonly installed software. Avoid using these kinds of applications.

Smiley

Many times you see smiley popup screen in your browser. Avoid clicking on that smiley screen.
