Microsoft Security Essentials –

Microsoft Security Essentials helps in guard against viruses, spyware, and other malicious software. It provides real-time protection for your PCs.

It runs quietly and efficiently in the background so you don't have to worry about interruptions or making updates.

Download link : http://windows.microsoft.com/en-us/windows/security-essentials-download

AVG –

AVG is very popular antivirus which detects and stops viruses, threats and malware.
