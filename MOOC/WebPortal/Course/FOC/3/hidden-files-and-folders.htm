If you want to hide some file and folders then you have to select file/folder, right click on the folder or file and click on properties. Files/folders properties dialog box will appear on screen. Then select the Hidden option and clicks apply.

files folders properties dialog box


To view the hidden files and folder, on windows explorer clicked on view-> folder options. Following dialog box will appear on screen.

View folder options dialog box


In folder options dialog box, select “Show hidden files and folders” and clicked on apply. After selecting this property, hidden files/folders will be visible to users.
