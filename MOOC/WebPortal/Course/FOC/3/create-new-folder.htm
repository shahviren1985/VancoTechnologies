  • Click on Windows start button, and then click on My Documents
  • Under File, click New and select Folder
  • A new folder is displayed with the default name as “New Folder”.
  • Type a name for the new folder, and then press ENTER key.
  • A new folder can also be created by right-clicking a blank area in a Window or on the desktop, pointing to New, and then clicking Folder
