All Programs : Displays a list of installed program

My Documents : Displays a list of documents, which could be used by the current user

My Recent Documents : Displays a list of recently used document by the current user

My Picture : Displays a list of picture/photos used by the current user

My Music : Displays a list of music/songs used by the current user

Control Panel : Displays a list of utility to configure the computer system and install software and hardware

Printers and Faxes : Displays a list of currently installed printers and faxes in the current computer system

Help and Support : Start the Windows 7 help and support program to find how to do a tasks in windows 7

Search : Helps the user to find the files and folder

Run : Start an application program

Log Off : Log off the currently logged in user

Turn Off : Computer closes the currently opened program, log of the current user and switch off the computer system
