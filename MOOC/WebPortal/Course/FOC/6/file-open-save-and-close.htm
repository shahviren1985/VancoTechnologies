Open file

You can open an existing file by several methods :

  • Go to Windows explorer and navigate to the path where your file is present. Double-click on the file.
  • Start MS Excel. Click on office button on the dropdown menu click 'open'. Select the file you want to open from the pop-up menu.

Save file

  • When you have finished your work on the file, you can save it by clicking on the “Save” icon at the top left corner.
  • Other way to save the file is - Click on office button> click on save at the dropdown menu.

When you are saving the worksheet for the first time follow the steps given below :

  1. Click office button
  2. Select “Save As” on the drop-down menu.
  3. On the pop-up menu select the location where you want to save the file.
  4. Type the file name
  5. Click on 'save' in the pop-up menu.
  6. You can change the file extension to any other format however the content of the file may get lost.

Keyboard shortcut to save a file: Press and hold down Ctrl key then press ‘S’ Key.


Close file

When your work is finished and it has been saved properly :

  • Select (Click) Close Command to close your file
  • Select (Click) Exit Excel Command to exit from MS Excel
