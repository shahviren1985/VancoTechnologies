You can edit you’re Excel worksheet data by any of the following ways :

  1. Select the cell and simply retype the correct entry.
  2. If you want only to clear the contents of the cell, select the cell and press Delete key.
  3. To bring back the previous entry, either click on Undo button on standard Toolbar or select Edit > Undo command or use keyboard shortcuts CTRL+Z.
