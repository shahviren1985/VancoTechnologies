Select File > Page Setup from the menu bar to format the page, set margins, and add headers and footers.

Page

The page option allows you to set the paper size, orientation of the data, scaling of the area, print quality, etc.

  • Select the Orientation under the Page tab in the Page Setup window to make the page Landscape or Portrait.
  • The size of the worksheet on the page can also be formatted using Scaling.
  • To force a worksheet to print only one page wide so that all the columns appear on the same page, select Fit to 1 page(s) wide.

Page Setup


Margins

  1. Change the top, bottom, left, and right margins by selecting Margins from the page setup group of page Layout Tab.
  2. Enter values in the header and footer fields to indicate how far from the edge of the page this text should appear.
  3. Check the boxes for centering horizontally or vertically on the page.
  4. There are three predefined margin settings. You can choose from them or you can also customize the margins as shown by the following diagram.

Margin Setup


Header/Footer text

For worksheets, you can work with headers and footers in Page Layout view. For other sheet types, such as chart sheets or for embedded charts, you can work with headers and footers in the Page Setup dialog box.
