The ribbon is provided for different chart related options. Buttons on the Insert tab of the ribbon are just the starting point to make a chart.

  • The Layout tab is used to add all sorts of elements related to chart or change the way they are shown in the chart.
  • The format tab is used to apply special effects which were not available in the earlier versions of the Excel, such as the bevel effect etc.

Following steps are given to draw a Chart

  1. Enter data in the work sheet

    2. worksheet with sample data


  2. Select data range: By using the mouse highlight the range of data you want to represent in the chart

    3. highlight data to create chart


  3. Click Insert Tab and select a chart type from the chart group.

    4. add charts


  4. Select the sub type of chart.

    5. Type of charts


    bar chart sample


  5. Select the Title of the chart
    1. To give a title to a chart, click on the chart. Now you can see layout tab available. Click on Layout tab.
    2. Choose(click) on chart title option available in the Label group

      3. chart title option


      chart title sample


    3. Click on the chart title and write a title.

      4. chart title output


  6. Give a name to X-Axis. Click on Layout tab. Then select Axis Titles from Labels Group. Select Primary Horizontal Axis Title, as shown below.

chart title option


chart title – below axis


Now, click on the Axis Title and write an X-axis title.


chart title at the bottom of axis


Follow the same steps to give a title to Y-axis.
